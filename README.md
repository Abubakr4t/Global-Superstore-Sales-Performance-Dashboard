# Global-Superstore-Sales-Performance-Dashboard
Interactive Power BI Dashboard analyzing global sales, profit, and category performance using DAX, KPIs, and responsive visuals.

# An Interactive Power BI Dashboard by Muhammad Abubakar

# Project Overview

This Power BI dashboard provides a comprehensive analysis of global sales performance across regions, product categories, and years.
It enables decision-makers to quickly assess sales trends, profit margins, and regional performance through interactive visuals and DAX-driven KPIs.

# Dashboard Goals

Analyze total Sales, Profit, and Quantity over time

Compare regional performance (with slicers for country/region)

Explore product categories and subcategories

Review yearly performance trends

Enable users to slice and filter data dynamically

# Dashboard Pages
# Page	Description
Overview	Shows total KPIs (Sales, Profit, Quantity) and trends.
Regional	Focused on region and country-level insights using slicers.
Categorical	Category and subcategory analysis for product insights.
Yearly	Year-wise comparison of sales and profit with time filters.


# Data Model

Fact Table: Orders (Sales, Profit, Quantity, Discount)

Dimensions: Customer, Product, Region, and Date
See details in Data_Model.md
